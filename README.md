Unsupervised analysis (clustering and network analysis) on biomarker data.
## Preliminaries

- Python, Jupyter, [Pipenv](https://pipenv.pypa.io/en/latest/)
- R

## Installation

```
Rscript install.R
pipenv shell
pipenv install
jupyter notebook
```
